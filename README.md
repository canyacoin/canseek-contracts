# CanSeek
CanSeek is a decentralized recruitment smart contract which allows employers and recruiters to interact freely.

### Install
```
npm install -g truffle
npm install -g ganache-cli
```

### Run locally
```
ganache-cli
truffle compile && truffle migrate
```

### Test
```
truffle test
```
